NodeJS-MySQL-API

A plane and basic API with MySQL and NodeJS.


**install modules**:
```
npm install
npm i --save-dev @types/express
npm i --save-dev @types/morgan
npm i --save-dev @types/cors
npm run build
npm run dev
```

**Importing DB:**
```
mysql -u root -p
CREATE DATABASE ng_games;
mysql -u root -p ng_games < data-dump.sql

```
